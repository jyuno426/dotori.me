# 피드백 시트 1

## 피드백 환경

- 모바일 사이즈에 대한 피드백임.
- 모바일에 대한 피드백 이지만, 다른 사이즈도 적절하게 변경 요망.

### 디자인 테마

- 금융 앱은 밝은 화이트 톤으로 신뢰감을 줘야한다는 의견이 있음.
- 지금은 다크 테마가 기본인데, 기본을 라이트 테마로 두고 사용자가 원하면 다크 테마로 전환 가능하면 좋겠음. (시스템 테마 자동 적용 x)

### 레이아웃

- 로그아웃 버튼이 헤더에 있는데, 설정(프로필) 페이지로 옮겨주면 좋겠어.
- 모든 곳에서 사용 가능한 loading overlay 있으면 좋겠어. 지금은 페이지 마다 로딩 문구가 따로 있는 것 같은데, 로딩 전환할 때 마다 UI가 끊겨서 너무 어색해.
- 헤더 양쪽 간격이 컨텐츠(바디) 영역 간격이랑 불일치 해. 동일하게 맞춰줘.
- 메뉴의 도토리 로고/텍스트 누르면 대시보드로 이동시켜줘.
- 헤더 프로필 영역 누르면 설정으로 이동시켜주면 어때? 설정이 프로필 관리 페이지 맞지?
- 메뉴 하단의 "한 알씩 모아, 단단한 내일로" 문구 영역 제거해줘.

### 로그인 화면

- "한 알씩 모아, 단단한 내일로" 이 문구 보다, 이 앱이 어떤 서비스 인지 나타내는 문구로 변경되면 좋겠음.

### 사용자 프로필

- 이름 변경이 가능하면 좋겠어

### 대시보드 화면

- "다계좌를 ~ 시작하세요." 문구가 너무 어려운 것 같아. 초보자도 쉽게 이해할만 한 쉬운 문구로 대체할 수 있을까? 쉬운 온보딩!

### 새 포트폴리오 화면

- placeholder에 "예: " 이 부분은 없어도 될 것 같아.
- 필수 선택 여부를 빨간 별표 `*`로 구분하면 어떨까?

### 개별 포트폴리오 화면

- 지금은 종목 추가를 계좌가 최소 1개 있어야 할 수 있어. 계좌가 없어도 설정할 수 있어야 하지 않을까?

### 종목 설정

- 종목 추가는 직접 입력 말고, 항상 검색으로 추가할 수 있도록 할래.
- 종목 검색 결과가 항상 empty야 무슨 이슈지?
- 자산군은 사용자가 직접 추가할 수 있게 하면 좋겠어.
- 종목의 현재가 입력창은 제거 해줘. cron으로 자동 수집할거야.
- 종목 리스트와 비중 설정 리스트가 별도로 구분되어 있는데.. 하나로 합칠 수 없나? 종목 추가 창에서 입력 가능해도 좋고. 엑셀처럼 한번에 관리가능한 최적의 ux면 좋겠어.


### 새 계좌 화면

- 포트폴리오 > 계좌 유형 > 증권사 > 걔좌 별칭 > 소유자 순으로 배치되면 좋겠어.
- 증권사는 필수로, select box 형태이면 좋겠어.
- 계좌 별칭은 선택으로 했으면 좋겠어.

### 각 계좌 기록 화면

- 기록을 2개 추가했는데, 최신 1개 밖에 안보여. 최신 기록 순으로 리스트 형태로 보여야 할 것 같아.
- 이미 입력한 기록은 삭제만 되고 수정은 안되는 것 같아. 수정도 되게 할 수 있을까?
