# 피드백 시트 2

## 피드백 환경

- 모바일 사이즈에 대한 피드백임.
- 모바일에 대한 피드백 이지만, 다른 사이즈도 적절하게 변경 요망.

### 레이아웃

- [x] 각 페이지의 개별 로딩 컴포넌트 제거하고, 공통 로딩 오버레이로만 표시.
- [x] 모바일 햄버거 메뉴 좌측 패딩어 너무 커. hover 영역에 대해 맞추어져 있는 것 같은데, 햄버거 메뉴 아이콘과 컨텐츠 패딩이 맞아야 해.
- [x] 데스크탑에서 헤더와 좌측 사이드바가 같이 있어서 이상해보여. 헤더 제거하면 어때? 프로필은 사이드바에 넣고?

### 종목 설정

- [x] 종목 삭제할 때 로딩 오버레이 표시해줘.
- [x] 종목 추가하는게 불편해. 검색이 오래걸리고, 하나 추가할 때 마다 로딩 대기해야 하고.
- [x] 비중 입력할 때 0 기본값이 있어서 그런지, 9를 입력하면 09 이런식으로 작성되는데, 보기 싫어.
- [x] 기본 자산군은 (선진국/신흥국 x 주식/채권) 4가지 조합과 대체 자산 1개, 현금성 자산 1개 이렇게 6개로 구성해줘.
- [x] 자산군별 종목 분포에 unknown으로 표시되는 이슈가 있어.
- [x] 자산군별 종목 분포에, 각 자산군 내의 개별 자산 분포도 표시할 수 있을까?
