## 3. 프리셋별 종목 매핑 예시

> **주의**: 아래는 *예시*이며, 도토리는 사용자에게 *카테고리별 후보 중 직접 선택*하게 한다. 자동 결정 X.

### 3.1 글로벌 60/40 — 예시 매핑

```
한국 주식 10%      → KODEX 200 또는 TIGER 200 중 1개
미국 주식 30%      → TIGER 미국S&P500 또는 KODEX 미국S&P500 중 1개
선진국 주식 15%    → TIGER 유로스탁스50(합성 H) + TIGER 일본니케이225 분할
신흥국 주식 5%     → KODEX 신흥국MSCI(합성) 등 중 1개
한국 채권 20%      → KODEX 국고채10년 등 중 1개
미국 채권 20%      → ACE 미국30년국채액티브(H) 또는 TIGER 미국채10년선물 중 1개
```

### 3.2 K-올웨더형 — 예시 매핑

```
한국 주식 10%      → KODEX 200 또는 TIGER 200
미국 주식 12%      → KODEX 미국S&P500 또는 TIGER 미국S&P500
선진국 주식 5%     → TIGER 유로스탁스50(합성 H) 등
신흥국 주식 3%     → KODEX 신흥국MSCI(합성) 등
한국 채권 25%      → KODEX 국고채10년·30년 등
미국 채권 25%      → ACE 미국30년국채액티브(H) 등
금 12%             → ACE KRX금현물 또는 KODEX 골드선물(H)
대안 8%            → TIGER 리츠부동산인프라 등
```

(영구포트폴리오·30대 성장형 매핑은 같은 방식으로 도출 — 사용자가 카테고리에서 선택)

## 4. 절세계좌 적합성 (참고)

> **주의**: v7 §4 법적 안전지대 — *"이 종목을 이 계좌에서"* 는 자문 영역 가까움. 도토리는 *"세금 효율 일반 가이드"*로만 노출.

| 자산군 | 일반 권장 계좌 | 이유 |
|---|---|---|
| 한국 주식 ETF | ISA·일반 위탁 | 이미 거래세 면제, ISA 비과세 한도 활용 |
| 해외 주식 ETF | 연금저축·IRP | 매매차익 15.4% 과세 → 과세이연 효과 큼 |
| 채권 ETF | 연금저축·IRP | 분배금 과세 → 과세이연 효과 |
| 금 (현물형) | 일반 위탁 | 분리과세 적용 가능 |
| 금 (선물형) | ISA | 매매차익 과세 → 비과세 한도 유리 |

> 이 표는 *교양 콘텐츠*로 노출. *"당신의 X 종목을 Y 계좌로 옮기세요"* 같은 1:1 자문 카피는 금지.

## 5. 데이터 모델 (구현 시)

```typescript
// src/data/portfolio-presets.ts (제안)

export interface AssetCategory {
  key: string;           // "kr-stock", "us-stock", etc.
  name: string;          // "한국 주식"
  description: string;
}

export interface PortfolioPreset {
  key: string;           // "permanent", "k-allweather", "global-60-40", "growth-30s"
  name: string;          // "영구포트폴리오"
  riskLevel: 1 | 2 | 3 | 4 | 5;  // risk-profile.md 매핑
  weights: { [categoryKey: string]: number }; // 합 100
  source: string;        // "해리 브라운 (1981)"
  description: string;   // 결 카피 (정상화 톤)
}

export interface ETFCandidate {
  ticker: string;        // "KODEX 200"
  isin: string;          // "KR7..."
  category: string;      // categoryKey
  ter: number;           // 0.0015
  manager: string;       // "삼성자산운용"
  exchangeCode: string;  // KRX
  notes?: string;
}
```

## 6. 데이터 운영

### 6.1 마스터 데이터 출처

- ETF 정보: KRX (Open API + ETF 마스터 CSV)
- TER: 운용사 공시 + KRX 정보
- 분기 1회 검수 (TER 변동·신규 상장·상장 폐지)

### 6.2 신규 ETF 진입 기준

§2 선정 기준 4개 모두 충족 시 추가. 1개 자산군당 *최대 5개*.

### 6.3 폐지 ETF 처리

- 폐지된 ETF는 후보 리스트에서 즉시 제거.
- 사용자 보유 종목인 경우 별도 안내 ("이 종목은 폐지 예정이에요").
