# 한국 절세계좌 자산배분 SaaS 사업 전략 문서

> **요약**: ISA·연금저축·IRP를 직접 굴리는 DIY 투자자를 위한 다계좌 통합 자산배분·리밸런싱·세금 최적화 SaaS. 주말 사이드 프로젝트 규모로 **6개월 내 MRR 100만원** 달성을 목표로 하는 실전 전략.

---

## 1. 사업 개요

### 1.1 한 줄 정의

**"한국 절세계좌 3종(ISA·연금저축·IRP) 전용 자산배분·리밸런싱·세금 최적화 도구"**

### 1.2 타겟 고객

- ISA·연금저축·IRP를 **동시에** 운용하는 직장인 DIY 투자자
- 올웨더·영구포트폴리오·60/40·SCHD 배당 등 **자기 전략을 가진** 투자자
- FIRE·파이어족·조기 은퇴를 준비하는 2030·40대
- 스노우볼72·김성일·강환국 콘텐츠 소비 경험자

### 1.3 핵심 지표 목표 (6개월)

| 지표 | 목표 |
|---|---|
| MAU | 5,000~10,000 |
| 유료 구독자 | 100~200명 |
| MRR | 100만원 이상 |
| 유료 전환율 | 2~3% |

## 2. 시장 기회

### 2.1 절세계좌 시장 규모

- **ISA**: 2025년 11월 기준 가입자 **719만명**, 가입액 46.5조원. 월평균 11만명씩 순증.
- **연금저축**: 2024년 말 **764만명**, 적립금 178.6조원. 연금저축펀드는 전년 대비 **+37.9%** 폭증.
- **IRP**: 2025년 3분기까지 적립금 **+23%**, 증권사 IRP는 **+31%**.
- **MZ 중심 폭발 성장**: 20대 연금저축 가입 **+35%**, ISA 투자중개형 20·30대 비중 40.7%.

### 2.2 실존하는 타겟 풀

세 계좌를 동시에 운용하는 직장인은 최소 **300~500만명**으로 추정. 이들은 세액공제 구조상 계좌를 나눠 써야 하지만, **증권사 앱은 계좌별로 쪼개져 있어 전체 자산배분 비중을 볼 수 없음**.

### 2.3 페인포인트 (커뮤니티 발언 기반)

- *"국내장은 공모주로 여러 증권사, 해외는 또 따로… 통합 관리가 안 된다"*
- *"매달 전체 수익률을 엑셀로 수기 계산한다"*
- *"시세는 30분 지연, 배당금은 수기 입력"*
- 구글 스프레드시트·엑셀에 의존하는 사용자가 압도적 다수

### 2.4 커뮤니티 규모

- 짠돌이부자되기 카페 **30만명**
- 거북이투자법 **52.7만명**

### 3.2 든든 (dndn) 분석

**본질**: 투자일임업·투자자문업 라이선스를 가진 제도권 금융회사 (AUM 3,300억, 2024년 1월). 김성일 대표 취임. 에버그린(올웨더 일임)·오로라(동적 일임) 두 전략 제공. 수수료는 AUM 기반 0.3~1%/년.

**구조적 한계 (= 우리의 기회)**:
1. **증권사 락인**: KB·NH·삼성·대신증권에서만 작동. 키움·미래에셋·토스증권 사용자 배제.
2. **전략 고정**: 2개 알고리즘만. 영구포트폴리오·60/40·김단테·SCHD 등 커스텀 전략 불가.
3. **통합 부재**: 든든에 맡긴 계좌만 보임. 다른 증권사·은행 계좌는 통합 불가.
4. **수수료 역설**: 1억 투자 시 연 30~100만원. 월 구독 1만원(연 12만원) 대비 10배 차이.

**경쟁 전략**: 정면충돌 불가(라이선스 요건만 자기자본 15억). **"같은 링에 오르지 않고 다른 카테고리에서 이긴다."**

```
든든  = "자율주행" (맡기세요)
우리 = "수동 기어 + 고성능 계기판" (당신이 운전, 우리가 계기판)
```

### 3.3 스노우볼72 분석

**본질**: 글로벌 금융기관 펀드매니저 출신이 만든 **자산배분 백테스트 플랫폼**. 정적/동적 자산배분 백테스트, 전략 비교·스크리너, 커뮤니티 랭킹, 리밸런싱 이메일 알림 제공.

**결정적 관찰**: **"과거 중심 연구 도구"**. 사용자가 전략을 *탐색·비교·검증*하는 단계까지 강력하지만, **"오늘 내가 실제로 뭘 해야 하는가"는 비어 있음**.

**투자자 워크플로우로 본 분업**:

| 단계 | 영역 | 담당 |
|---|---|---|
| 1. 학습·전략 선택 | 어떤 전략을 따를 것인가 | **스노우볼72** |
| 2. 실행·집행 | 오늘 몇 주를 매수할 것인가 | **우리** |
| 3. 추적·편차 관리 | 목표 비중에서 얼마나 벗어났나 | **우리** |
| 4. 세금 최적화 | ISA 한도, 이체 시점 | **우리** |
| 5. 은퇴·FIRE 시뮬레이션 | 언제 경제적 자립 가능한가 | **우리** |

**경쟁 전략**: **보완재 포지션**. "스노우볼72로 전략을 고르고, [내 서비스]로 실행하세요" 서사. 스노우볼72 사용자는 이미 교육된 얼리어답터라 가장 효율적인 타겟.

**구체적 차별화 축 5가지**:
1. **"지금 무엇을 사야 하는가" 구체성**: 스노우볼72는 목표 비중만 제시, 우리는 실행 단위까지.
2. **다계좌 통합 뷰**: 스노우볼72는 전략 단위, 우리는 계좌 단위 통합.
3. **한국 세법 특화**: ISA 비과세 200만원, 1,500만원 분리과세, ISA→IRP 10% 추가 세액공제.
4. **모바일 퍼스트**: 스노우볼72는 데스크톱 중심. 일 단위 사용 빈도 구조적 차이.
5. **초심자 접근성**: 모멘텀·카나리아 용어 없는 UX.

## 4. 법적 안전지대

### 4.1 규제 없이 가능한 영역

자본시장법 제7조 제3항: *"불특정 다수에게 발행되는 간행물·통신물 등을 통한 조언은 투자자문업이 아니다"*. 대법원 2018도4413: 특정성 기준은 "면담·질문으로 개별 투자목적·재산상황·경험이 반영되는가".

**합법적 운영 범위**:
- 사용자 수동 입력 기반 **리밸런싱 계산기**
- 공식 기반 자산배분 **제안** (나이×10%, 60/40 등)
- 과거 데이터 **백테스팅**
- ETF/펀드 정보 **비교 표시**
- 사용자 설정 기반 **알림** (비중 초과 등)
- 무료 서비스의 **애드센스·일반 광고**

### 4.2 반드시 피해야 할 영역

- **특정 종목 매수/매도 시점 조언** → 유사투자자문업 신고 필요 (2024.8.14 개정으로 양방향 유료채팅방은 금지, 3년 징역/1억 벌금)
- **1:1 맞춤 자문** → 투자자문업 등록 (자기자본 2.5~15억)
- **증권사 계정 스크래핑** → 마이데이터 시행 이후 위법
- **금융상품 청약 유인 포함 추천** → 금소법상 중개업

### 4.3 실전 결론

**"사용자가 수동으로 보유 종목을 입력하거나 공공 데이터(KRX/DART)만 쓰는 계산·시각화·시뮬레이션 도구 + 구독 모델"**은 라이선스 없이 **완전히 합법**. 증권사 계좌개설 제휴링크(CPA)는 "계좌"가 금융상품이 아니므로 일반적으로 허용 (단, 특정 펀드·ETF 추천과 청약 유인 결합은 회피).

## 5. 제품 구성

### 5.1 데이터 입력 전략

**핵심 원칙**: 마이데이터·증권사 API 연동 불가는 개인 개발자의 공통 제약이며 약점이 아님. 자산배분 투자자에게 **"실시간 추적" 자체가 필요 없음**.

#### 레이어 1 — 종목 수량 입력 + 시세 자동 (기본)

사용자는 **종목코드와 보유 수량**만 입력, 가격은 공개 데이터로 자동 갱신.

- 국내: KRX 마켓데이터 / pykrx / 네이버·다음 금융 시세
- 해외: Yahoo Finance (yfinance) / Alpha Vantage
- 환율: 한국은행 ECOS API
- 배당: DART 공시 또는 종목별 크롤링

**장점**: 개인 개발자 즉시 구현 가능, 법적 이슈 0, 주간·월간 관리 빈도에 충분.

#### 레이어 2 — 스크린샷 OCR (모바일 최적화)

사용자가 증권사 앱 "보유 종목" 화면을 **스크린샷 1장**으로 올리면 자동 파싱.

- Claude Vision API / GPT-4V / Google Cloud Vision으로 한국어 OCR
- 한국 증권사 앱 UI는 다르지만 "종목명 + 수량 + 평가금액" 구조는 동일
- 비용: 스크린샷 1장당 API 비용 ~10원. 월 1회 사용 시 사용자당 월 10원 수준
- OCR 후 이미지는 즉시 삭제, 서버 저장 금지 (프라이버시)

**이 기능의 의미**: 스노우볼72·더리치 어디에도 없는 경험. **"사진 한 장으로 포트폴리오 업데이트"**가 모바일 퍼스트 경험의 상징.

### 5.2 핵심 기능

#### (1) 다계좌 통합 자산배분 대시보드

- ISA·연금저축·IRP·일반계좌를 **계좌 유형별 구분**하여 통합
- 올웨더·영구포트폴리오·60/40·커스텀 **타깃 비중 대비 현재 편차** 시각화
- 사용자 정의 전략 템플릿 저장
- IRP는 주식형 70% 상한 자동 반영

#### (2) 리밸런싱 계산기

- 편차 ±5%p 초과 시 알림
- 실행 단위까지 계산: *"TIGER 미국S&P500 3주 매수, KODEX 국고채10년 2주 매도"*
- **세금 고려 리밸런싱**: ISA 비과세 200만원 한도 우선 사용, 연금계좌 과세이연 유지 우선 매도

#### (3) 세금 최적화 도구

- ISA 비과세 200만원 한도 실시간 잔여 추적
- 연금저축 1,500만원 분리과세 한도 시뮬레이션
- ISA 만기 시 IRP 이체 시 10% 추가 세액공제(최대 300만원) 계산기
- ISA 만기 연장 vs 재가입 vs 이체 3시나리오 비교

#### (4) FIRE·은퇴 시뮬레이터

- 현재 자산·저축률·예상 수익률 입력 → **은퇴 시점 역산**
- ISA·연금 세제혜택 반영
- 4% 룰·7% 룰 비교
- 연금수령 전략 최적화

### 5.3 UX 철학 — "덜 보게 만드는 앱"

**배경**: 자산배분 투자는 매일 보는 것이 오히려 해로움. 구루들의 일관된 원칙: *"자주 보지 마라"*. 표준 리밸런싱 주기는 분기 1회 또는 연 1회, ±5%p 편차 기준.

**설계 원칙**:
- 홈 화면 첫 줄: *"이번 달 리밸런싱 필요 없음 ✓"* (편차가 임계값 이내일 때)
- 푸시 알림: **편차 ±5%p 초과 시에만** 발송, 평소엔 침묵
- 월급일 리마인더: *"오늘 월급일 같네요. 적립 매수 후 수량만 업데이트 해주세요."*
- 분기 리포트: 자동 생성 이메일 (3개월 수익률·비중 변화·세금 사용 현황)
- **월 사용 빈도 2~4회 목표**

**이것이 차별화 포인트**: 증권사 앱이 호가 변동·뉴스·체결 알림으로 트레이딩을 자극하는 반면, 우리는 **"투자 FOMO로부터 해방시켜주는 앱"**.

### 5.4 마이데이터 불가를 강점으로 재프레임

1. **프라이버시**: *"계좌 비밀번호도, 공동인증서도, 마이데이터 동의도 필요 없습니다."* 로컬 저장(IndexedDB) + 선택적 E2E 암호화 클라우드 동기화.
2. **증권사 독립성**: 증권사 정책 변경·장애에 무관하게 작동.
3. **오프라인 동작**: 마이데이터 API 지연에 무관.

| 플랜 | 가격 | 포함 기능 |

| Free | ₩0 | 3계좌 · 기본 리밸런싱 · 수동 입력 |

**벤치마크**: 토스프라임 월 5,900원, 리디셀렉트 월 4,900원. 투자 관련 유료 콘텐츠는 월 9,900원~35,000원 (유목민 채널). **7,900원은 합리적 중간값**.

### 6.2 MRR 100만원 달성 경로

- 전환율 2% 가정
- 7,900원 기준 **130명** → MRR 약 100만원
- 필요 MAU: **6,500명**
- 증권사 계좌개설 CPA (5,000~50,000원/건) 추가 시 **월 150만원 이상** 가능

### 6.3 보조 수익원 (점진 도입)

- **증권사 CPA 제휴**: 키움·미래에셋·한국투자·토스증권 비대면 계좌개설 이벤트
- **네이버 프리미엄콘텐츠 채널**: "3계좌 자산배분 실전편" 월 3,900~9,900원 구독
- **애드센스**: SEO 트래픽 유입 페이지에만 적용 (메인 앱은 광고 없음 정책)

## 10. 최종 결론

### 10.1 왜 6개월 내 MRR 100만원이 가능한가

- **시장**: ISA 719만 + 연금저축 764만, 매월 11만명 신규 가입. 명백한 거대 수요풀.
- **경쟁 공백**: 한국 절세계좌 특화 다계좌 자산배분 도구는 한국에도 해외에도 없음. 블루오션의 정확한 좌표.
- **법적 여유**: 수동 입력 기반 계산·시각화 도구는 라이선스 불필요.
- **단가 수렴**: 130명 × 7,900원 = 100만원. 커뮤니티 규모(수십만명) 대비 도달 가능.
- **Claude Code 속도**: 주말 리소스로 12주 MVP 가능.

### 10.2 핵심 차별화의 쐐기

**"한국 절세계좌 3종 통합 × 자산배분 × 세금 최적화 × 모바일 퍼스트"**

이 네 가지 축의 교집합에는 현재 아무도 없음. 각 축을 단독으로 파는 서비스(든든·스노우볼72·더리치·뱅크샐러드)는 있지만, 네 축의 교차점을 점유하는 서비스는 부재.

### 10.3 피벗 이후의 첫 일

1. 현재 자산배분 툴의 UI에서 **"계좌 유형(ISA/연금저축/IRP/일반)" 구분 필드 추가**
2. 랜딩페이지 카피 "당신의 세 계좌를 한눈에"로 교체
3. 거북이투자법·짠돌이부자되기 카페에 메이커 스토리 초안 작성
4. 네이버 프리미엄콘텐츠 채널 개설

Parqet(독일)이 1년에 MRR €15K, ProjectionLab이 2년에 MRR $10K를 달성한 플레이북이 **한국 절세계좌 시장에서는 더 큰 규모로 재현 가능**. 단, 먼저 **"한국 절세계좌 전용"이라는 포지셔닝의 칼날을 세우는 것**이 모든 것의 전제조건.
